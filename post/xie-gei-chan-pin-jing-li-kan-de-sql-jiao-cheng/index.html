什么是sql

后端服务一般需要存储用户数据，数据的组织形式可以大致想像成excel表格，每一行是一条记录数据，有很多列，代表不同字段信息。比如表格student是存储学生信息（包括学号，姓名，性别，年龄）。

sql的命令主要包括select(查询数据)，update(更新数据)，delete(删除数据)等，产品经理使用sql更多是查询数据，也就是select命令，对于记录学生数据的表格A，我们可以一句查询的sql从中读出来我们想要的数据，比如

select * from student where number = 1002;

就可以把学号是1002的学生信息查出来。我们来解析下这句sql：其中select * from student的含义是从student这张表格里查询(select)所有(*)数据，很语义化，其实蛮好理解的；接下来是where number = 1002，sql语句中where用来表示过滤条件，因为我们要查学号是1002的学生信息，所以过滤条件就是“学号等于1002”。
